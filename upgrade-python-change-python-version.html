Upgrade Python / change python version

Raspbian OS comes with with python but not latest version
if we want to use latest version , we need to install it seperately (and manually) and change variable python3 to run to the correct version
Installation is a little bit tricky because we need to compile the source code ourself

Download Python 3.12.0 source

Fist download latest python 3 version
Run this command to download the latest Python source code.

wget https://www.python.org/ftp/python/3.12.1/Python-3.12.1.tgz

Change default python version

Now whenever you want to execute command python3
it will look up in /usr/bin
if you want to use new version , along with keeping the system OS python
You can use alias to map python or python3 to any version you want

python = System python(old version)
python3 = latest version

now you can use python3 as lastest version and python as system os version
